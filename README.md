# Tampermonkey Universal AI Prompt Commands HR

Tampermonkey skripta za zamjenu kratkih naredbi gotovim AI promptovima u AI chatovima.

Skripta zamjenjuje samo točne naredbe pripremljenim promptovima. Običan tekst se ne mijenja. ChatGPT, Gemini, Claude, Copilot.

## Naredbe

- `HR1` — Prevedi celý poskytnutý text přesně do hrvatski.
- `HR2` — Sažmi poskytnutý text v hrvatskom podle významu a kontextu.
- `HR3` — Izradi v hrvatskom krátké tematické shrnutí dopisu přesně na jeden řádek.
- `HR4` — Prevedi poskytnutý text do jednoduché a srozumitelné němčiny na úrovni A2-B1.
- `HR5` — Oprav poskytnutý hrvatski text.
- `HR6` — Napiši krátkou, zdvořilou a úřední odpověď na tento dopis v jednoduché němčině úrovně A2-B1.
- `HR7` — Vysvětli jednoduchými slovy v hrvatskom, co text znamená.
- `HR8` — Získej z textu všechny důležité skutečnosti a uspořádej je v hrvatskom: osoby, organizace, adresy, data, lhůty, částky, čísla dokumentů, požadavky, rozhodnutí, povinnosti, dokumenty a další kroky.
- `HR9` — Sestav v hrvatskom jasný seznam kroků podle textu.
- `HR10` — Na základě textu napiš zdvořilý úřední dopis v jednoduché němčině úrovně A2-B1.

## Instalacija

Instalirajte Tampermonkey, otvorite poveznicu userscripta, potvrdite instalaciju i unesite prvu naredbu u AI chat.

[Userscript](https://raw.githubusercontent.com/1777maxim7771/hr_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js)

Skripta se instalira u proširenje Tampermonkey, a ne u GitHub ili stranicu. GitHub samo pohranjuje datoteku.

Tampermonkey → Dashboard → Utilities → Import from URL:

`https://raw.githubusercontent.com/1777maxim7771/hr_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js`
